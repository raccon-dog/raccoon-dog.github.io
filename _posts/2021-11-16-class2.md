## PRINT AND GET USER INPUT  

# PRINT 함수
- print() 함수는, 화면에 출력하는 함수다.


```python
sentence = '제 이름은 홍길동입니다. 나이는 25입니다.'
```


```python
sentence
```




    '제 이름은 홍길동입니다. 나이는 25입니다.'




```python
print(sentence)
```

    제 이름은 홍길동입니다. 나이는 25입니다.
    


```python
name = input('이름 입력 :')
age = int(input('나이 입력 :'))
print('제 이름은 {}입니다. 나이는 {}입니다.'.format(name,age))
```

    이름 입력 :221
    나이 입력 :2121
    제 이름은 221입니다. 나이는 2121입니다.
