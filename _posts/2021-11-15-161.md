```python
for i in range(1,10,2):
    print(f"3 x {i} = {3*i}")
```

    3 x 1 = 3
    3 x 3 = 9
    3 x 5 = 15
    3 x 7 = 21

# 168
1~10까지의 숫자에 대해 모두 더한 값을 출력하는 프로그램을 for 문을 사용하여 작성하라.

합 : 55


```python
x = 0
for i in range(11):
    x += i
print(f"합: {x}")
```

    합: 55
