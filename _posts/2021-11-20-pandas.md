PYTHON PROGRAMMING FUNDAMENTALS  


 Pandas 의 장점
- Allows the use of labels for rows and columns
- 기본적인 통계데이터 제공
- NaN values 를 알아서 처리함.
- 숫자 문자열을 알아서 로드함.
- 데이터셋들을 merge 할 수 있음.
- It integrates with NumPy and Matplotlib


```python
import pandas as pd 
```

## Arithmetic Operations on Pandas Series



```python

```


```python

```




    apples     10
    oranges     6
    bananas     3
    dtype: int64




    apples     15
    oranges    11
    bananas     8
    dtype: int64




```python
fruits['oranges'] = fruits['oranges'] - 2
```


```python

```




    apples     15
    oranges     9
    bananas     8
